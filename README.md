## Drag and Drop sortable lists with Rails & Stimulus JS

we used:
- stimulus
- ajax
- sortablejs
- acts_as_list gem
- rails webpacker:install:stimulus

Add this to the webpack config:

```javascript
environment.plugins.prepend('Provide',
  new webpack.ProvidePlugin({
    Rails: ['@rails/ujs']
  })
)
```
